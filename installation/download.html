Fuel をダウンロードする

Fuel は現在バージョン 1.1 です。次のリンクを使用して Fuel をダウンロードできます:

  • リリース
    • Download 1.1
    • Download 1.0.1
    • Download 1.0
  • リポジトリ
    • Fuel on GitHub
    • Fuel Core on GitHub
    • Docs on GitHub
    • Fuel Orm on GitHub
    • Fuel Auth on GitHub
    • Fuel Parser on GitHub
    • Fuel Oil on GitHub
    • Fuel Email on GitHub
  • ユーザ作成の Fuel パッケージ
    • GitHub 上の Fuel-packages アカウント
      あなたのパッケージを fork して fuel-packages に取り込むためには、このアカウントか、 Twitter の @fuelphp までメッセージを送ってください。

コマンドラインから Fuel の承認されたパッケージの 1 つをインストールするために Oil ユーティリティ を使用します。

コアに貢献する

Fuel はコミュニティ主導で開発されており、私たちは参加を歓迎しています。もし貢献に興味があれば、 私たちの コーディング標準 を読む時間を少し割いてください。 これらのガイドラインを読んでいない貢献は受け入れられません。読んでいれば、fork、commit、Pull Request することができます。 方法がわからない場合は: http://help.github.com/forking/ を参照してください。

パッケージに貢献する

他の Fuel ユーザに興味を持ってもらえるものが何かあるでしょうか? それは パッケージ を作成することです。 適切にコーディングされており、コーディング標準 に準拠していれば、 私たちはこのページにそれを掲載し、oil コマンドラインユーティリティを使用してインストールできるようにします。
