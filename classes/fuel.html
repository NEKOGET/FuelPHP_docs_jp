Fuelクラス

Fuelクラスには、Fuelフレームワークのコアメソッドが含まれています。 本ドキュメントでは、アプリケーションで利用可能な同クラスのpublicメソッドを纏めています。 ここに記していないメソッドはクラス内部でのみ利用されるものなので、利用してはいけません。

find_file($directory, $file, $ext = '.php', $multiple = false, $cache = true)

find_fileメソッドは、Fuelアプリケーションディレクトリ構造下にあるファイルを見つける時に使います。

本メソッドは、アクティブなリクエストインスタンスの下で設定されている全てのサーチパス、アプリケーションのパス、ロードされている全てのパッケージのパス、コアパスを（このパス順で）探査します。'add_path'メソッドを使って、独自に（manually）グローバルサーチパスを設定することも出来ます。

Static Yes
パラメタ
Param 初期値 解説
$directory 必須 探査したい相対ディレクトリ名
$file 必須 探査したいファイル名
$ext 
'.php'
 探査したいファイル拡張子。冒頭のドットは必須です！
$multiple 
false
 もしfalseならば、最初に見つけたファイル名が返されます。もしtrueならば、該当するファイル名すべてを、配列で返します。
$cache 
true
 もしtrueならば、以後の検索リクエストのために結果がキャッシュされます。falseならばキャッシュされません。
返り値 Mixed, 文字列か文字列の配列です。
例 
// viewsディレクトリにある 'test' 名ファイルを全て見つけます。
$views = Fuel::find_file('views', 'test', '.php', true);

list_files($directory = null, $filter = '*.php')

list_filesメソッドは、ロードされている全てのサーチパス（例えば、それらのパスが派生したファイル構造 cascading file systemをもっているような場合）内にある指定ディレクトリ下のファイル一覧を取得します。全サーチパス内のconfigファイルを全べて見つけ出そうとする時などに、便利なメソッドです。

Static Yes
パラメタ
Param 初期値 解説
$directory 必須 探査したい相対ディレクトリ名
$filter 
'*.php'
 探査時に使うファイル名フィルタ
返り値 配列
例 
// インストールされているFuelの中の全てのconfigファイルを見つけ出します。
$files = Fuel::list_files('config');

add_path($path, $prefix = false)

add_pathメソッドを使えば、グローバルサーチパスを付け加えることが出来ます。付け加えられたパスは、Fuel::find_file()によって、アプリケーション中のファイルの所在を突き止めるために利用されます。

Static Yes
パラメタ
Param 初期値 解説
$path 必須 グローバルサーチパス一覧に付け加えるパス。
$prefix 
false
 もしfalseならば、パスは、サーチパス一覧の後尾に付け加えられます。trueならば一覧冒頭に付け加えられます。
返り値 void
例 
// アプリケーションのmyfiles folderをグローバルサーチパス一覧の冒頭に付加します。
Fuel::add_path(APPPATH.'myfiles'.DS, true);

Static Yes

// 設定されているサーチパスを取得

load($file)

loadメソッドは、指定ファイルをロード（インクルード）し、その結果を返します。例えば、設定ファイルをロードする時に本メソッドは使われます。その場合、設定情報は配列で返されます。

Static Yes
パラメタ
Param 初期値 解説
$file 必須 ロードしたいファイル名
返り値 Mixed
例 
// あるPHP fileをロード
if (Fuel::load('myfile.php'))
{
	echo "MyFile was loaded successfully";
}

次の点に留意してください。もしインクルードされるファイルにreturn宣言が含まれていない(NOT)場合、ロードが成功すればload() は1を返し、失敗すれば0を返します!

add_module($name)

add_moduleメソッドは、指定されたモジュール（名） をクラスのプレフィックスとして登録します。

Static Yes
パラメタ
Param 初期値 解説
$name 必須 モジュール名 (アンダースコア無しの小文字プレフィックス)
返り値 文字列
Throws FuelException、指定した名前をもつモジュールが存在しない時。
例 
// 'mymodule'モジュールを登録
try
{
	$module_path = Fuel::add_module('mymodule');
}
catch (FuelException $e)
{
	// ううむ。モジュールが存在しない。なにかやってください！
}

module_exists($module)

module_existsメソッドはモジュールが存在するかどうかをチェックします。したがって、チェックしたいモジュールをあらかじめ付け加えておく必要はありません（The module you want to check does not have to be added first.）。

Static Yes
パラメタ
Param 初期値 解説
$module 必須 モジュール名 (アンダースコア無しの小文字プレフィックス)
返り値 mixed
例 
// 'mymodule'パスを返します。
$module_path = Fuel::module_exists('mymodule');

// unknownモジュールがないのでfalseを返します。
$module_path = Fuel::module_exists('unknown');
